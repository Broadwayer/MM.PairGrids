## <a name="english"></a>English

**WPF Custom Controls Library for Displaying Key-Value Collections (PropertyGrid Alternative)**
A library for easily displaying collections of key-value elements, including nested child elements. Similar in functionality to WinForms PropertyGrid, but implemented for WPF.
### Features
- Display key-value collections.
- Support for nested child elements.
- Flexible templates and data binding.
### Installation
1. Clone the repository and add the project to your solution:
   git clone https://github.com/yourusername/MM.PairGrids.git
2. Or build the DLL and reference it in your project.
### Examples
For usage examples, see the PairGridsTestApp1 project in the solution.

Benutzerdefinierte WPF-Steuerelement-Bibliothek zur Anzeige von Schlüssel-Wert-Sammlungen (PropertyGrid-Alternative)
Eine Bibliothek zur einfachen Anzeige von Schlüssel-Wert-Elementen, einschließlich verschachtelter untergeordneter Elemente. Funktionell ähnlich wie das PropertyGrid aus WinForms, aber für WPF implementiert.

- Unterstützung für verschachtelte untergeordnete Elemente.
